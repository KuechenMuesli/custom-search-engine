# How to set up

## Chrome 
To set this software up in chrome, you will need to add a new search engine and set it as your default.

1. To achieve this, open the chrome settings and navigate to "search engine"
   Alternatively, go directly to [chrome://settings/searchEngines](chrome://settings/searchEngines).
2. In the *Site search* section, click **Add**. 
3. A dialog will pop up, enter the following values:
	- Name: **paul-simon-search** (or whatever you want to name it)
	- Shortcut: (type anything, doesn't matter)
	- Url: `https://search.paul-simon.dev?q=%s`
4. Click **Add**
5. Click on the three dots next to the new entry
6. Click on **Set as standard**

Now every search in chrome will be redirected to [search.paul-simon.dev](https://search.paul-simon.dev) with the according query.

## Safari
Safari, by default, does not let you configure a custom search engine, so using an extension is required.

For example the extension *[Customize Search Engine](https://apps.apple.com/de/app/customize-search-engine/id6445840140)* may be used.
Follow the extension's instructions and type `https://search.paul-simon.dev?q=%s` into the field in the tab *standard search engine*.
